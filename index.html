How To Host a Site on GitHub

Zach Santarelli

Step 1: Opening Git Bash

Right-click on the folder that contains your website and select GitBash

Step 2: Starting Commands

Input the following commands into the CMD:

  1. git init
  2. git checkout -b gh-pages

Step 3: Commiting Changes

Commit changes by doing the following:

  1. git add filename
  2. git status
  3. git commit -m "message"
  4. git status

Step 4: Setting Up The Repo

To set up a repo, first navigate to GitHub.com and select "Add Repository".

  1. name the repository

Step 5: Linking to the repo

Input the following commands to the Git Bash CMD:

  1. git remote add origin https://github.com/username/reponame.git
  2. git push -u origin gh-pages

Done!

Your website should be online and working now!
